# BABA Wallet: The First Credits Blockchain Wallet

![Version](https://img.shields.io/badge/version-1.2-blue) ![Platform](https://img.shields.io/badge/platform-Android-green) ![License](https://img.shields.io/badge/license-Proprietary-lightgrey)

**BABA Wallet** is a secure, non-custodial mobile wallet designed specifically for the **Credits (CS)** blockchain network. It provides a simple, fast, and secure interface for managing your crypto assets without intermediaries.

> **Your Keys, Your Coins.** BABA Wallet is non-custodial, meaning your private keys never leave your device and are encrypted locally.

---

## 🚀 Key Features

* **Non-Custodial Security:** Full control over your private keys and mnemonic phrases.
* **Credits Network Native:** Optimized specifically for the Credits blockchain speed and architecture.
* **Asset Management:** Send and receive CS coins and tokens instantly.
* **Transaction History:** View detailed logs of your blockchain activity.
* **Multi-Language Support:** Fully localized for English, Persian (Farsi), Russian, and Spanish users.
* **Dark/Light Mode:** Designed for visual comfort.

## 🔐 Security Verification (Checksums)

To ensure the APK you downloaded has not been tampered with, you can verify its **SHA-256** fingerprint.

**Release Tag: v1.2**
```text
Filename: app-release.apk
SHA-256:  34203b016046ea78f6a5a9bc273438e8586546c542029f0abb728984ab7c51fa
```

*How to verify:*
* **Mac/Linux:** Open terminal and run `shasum -a 256 app-release.apk`
* **Windows:** Open PowerShell and run `Get-FileHash app-release.apk`

**Note:** The output code must match the SHA-256 line above exactly.
